## 2. 项目技术路线

```text
微信原生小程序 WXML
        ↓
Django REST API
        ↓
MySQL / SQLite
        ↓
AI API（DeepSeek / 通义千问 / OpenAI 等）
        ↓
咖啡知识库 Markdown / JSON / 简单向量检索
```

### 开发原则

- [ ] AI API Key 只放后端环境变量，不放小程序前端
- [ ] 小程序只请求 Django，不直接请求 AI 服务
- [ ] AI 返回结果尽量结构化 JSON
- [ ] AI 推荐必须匹配真实商品 `product_id`
- [ ] 优先完成闭环，不追求功能数量
- [ ] 每天保留开发记录，方便写复盘和简历

```json
  "user_input": "我想喝一杯冰的、低糖、不要牛奶的咖啡",

}
```

### Response

```json
{
  "success": true,
  "data": {
    "intent": "recommend_drink",
    "recommendations": [
      {
        "product_id": 12,
        "product_name": "冰美式",
        "reason": "符合冰饮、低糖、无奶需求",
        "options": {
          "temperature": "ice",
          "sugar": "low",
          "milk": "none"
        }
      }
    ],
    "follow_up_question": ""
  },
  "message": "推荐成功",
  "error_code": null
}
```

```text
调整建议：

3. 总时间拉长到 2:15-2:30
```
